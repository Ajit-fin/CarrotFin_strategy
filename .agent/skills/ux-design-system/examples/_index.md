# UX Design System — Example Inventory

> **Last updated:** 2025-04-15

---

## Status

This directory will contain annotated examples of screen compositions — illustrating how the AI assembles components for specific user contexts. Examples are added after component patterns are designed and validated.

## Example Template

When adding a new example, use this structure:

```markdown
# [Example Name] — [Screen] for [User Context Summary]

> **Created:** YYYY-MM-DD  
> **Illustrates:** [which patterns and composition rules]

## User Context
| Dimension | Value |
|---|---|
| Financial literacy | [1-5] |
| Risk tolerance | [conservative/moderate/aggressive] |
| Life stage | [description] |
| Trust level | [new/warming/trusting] |

## Composition
[Ordered list of components with brief rationale for each selection]

## What Would Change
[2-3 variants describing how a different user context produces a different composition]

## Related Patterns
[Links to pattern definitions in patterns/]
```

---

*Examples are illustrative, not canonical. Always label the user context that produced them.*
