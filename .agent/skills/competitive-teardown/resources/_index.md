## UX Gap Checklist

Use this for quick cross-competitor comparisons on specific dimensions:

| Dimension | Questions to Answer |
|---|---|
| **Onboarding friction** | How many steps before first value? Data points requested upfront? Progressive vs. front-loaded? |
| **Personalization signals** | What user data does the app collect? How visibly does it personalize? Segment-level or individual? |
| **AI integration depth** | Is AI visible? Chatbot? Copilot? Native to the experience? Or absent entirely? |
| **Navigation model** | Tab bar? Hamburger? Search-first? How many taps to core features? |
| **Recommendation quality** | Generic ("top funds") or contextual ("for your situation")? Evidence shown? |
| **Information density** | Sparse (consumer-friendly) or dense (power-user)? Adjustable? |
| **Trust mechanics** | How does the app earn trust? Social proof? Transparency? Credentials? |
| **Retention hooks** | What brings users back? Notifications? Reports? Streaks? Value delivery? |
